# sakib-os
My portfolio as a live operating system.

## Project Structure

```text
sakib-os/
│
├── index.html                    # Main HTML entry point
├── README.md                     # Project documentation
│
├── css/
│   └── styles.css                # Complete OS styling
│
├── js/
│   │
│   ├── core/
│   │   ├── main.js               # Boot sequence & initialization
│   │   ├── os-core.js            # System core & version information
│   │   ├── system-state.js       # Global state manager
│   │   ├── window-manager.js     # Window management system
│   │   ├── command-system.js     # Command interpreter & search
│   │   ├── sound-engine.js       # Web Audio API sound engine
│   │   ├── theme-engine.js       # Dynamic theme management
│   │   └── icons.js              # Icon registry & helpers
│   │
│   └── apps/
│       ├── about.js              # About Me
│       ├── resume.js             # Resume Viewer
│       ├── github.js             # GitHub Dashboard
│       ├── linkedin.js           # LinkedIn Profile
│       ├── youtube.js            # YouTube Channel
│       ├── contact.js            # Contact Terminal
│       ├── game.js               # Game Hub
│       └── vault.js              # Sakib's Vault
│
└── assets/
    ├── profile.jpg               # Profile Photo
    ├── resume-preview.png        # Resume Preview
    └── wallpaper.jpg             # Desktop Wallpaper
```
